# Protective Clarity Theme

A custom theme for Anonymize.Education - Education Data Protection

## Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| **Guardian Teal** | #0d9488 | Primary brand color, CTAs, trust elements |
| **Deep Knowledge** | #134e4a | Headers, dark accents, footer |
| **Nurture Green** | #5eead4 | Highlights, success states, growth |
| **Safe Cream** | #f0fdfa | Backgrounds, breathing space |
| **Warm Amber** | #f59e0b | Accents, attention, warmth |
| **Slate Wisdom** | #475569 | Body text, secondary content |

## Typography

**Headers:** Work Sans Bold
- Clean, geometric, modern yet approachable
- Suggests precision without coldness

**Body Text:** Work Sans Regular
- Excellent readability
- Professional but friendly

**Accent/Labels:** IBM Plex Mono
- Technical credibility
- Data/security context

## Visual Elements

- Concentric circles (layers of protection)
- Soft rounded rectangles (safe containers)
- Gradient meshes (flowing, organic security)
- Dot patterns (data points being organized)
- Shield motifs (subtle, abstracted)

## Best For
Educational institutions, data protection services, privacy-focused products, GDPR compliance, child safety technology
