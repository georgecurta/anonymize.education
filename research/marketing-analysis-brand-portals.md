## Executive Summary

**Anonym.legal Brand Portals** represent a premium vertical strategy deploying 4 specialized portals targeting high-value market segments with zero-knowledge PII protection. All portals share the same backend infrastructure (Microsoft Presidio + ZK security layer), enabling capital-efficient scaling across verticals.

### The 4 Brand Portals

| Portal | Target Vertical | Key Differentiator |
|--------|-----------------|-------------------|
| **blurgate.legal** | Image anonymization | OCR + Tesseract + Presidio |
| **anonymize.dev** | Developers | MCP Server for AI IDEs |
| **cloak.business** | Enterprise GDPR | Transparent audit trails |
| **anonymize.education** | Education | Student data protection |

### Strategy: Premium Vertical Positioning

- **Higher ACV**: Zero-knowledge justifies 2-5x premium pricing vs. standard anonymization
- **Vertical Focus**: Each portal addresses specific compliance needs (GDPR, HIPAA, FERPA, PCI-DSS)
- **Scalable Model**: Same backend infrastructure serves all portals
